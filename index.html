Creating tables simply and quickly in R and knitr

This webpage is a single link to R code I've been testing to create tables of results simply and quickly. There are amazing packages in R to create great looking tables of results, publishable as html, docx or pdf. I am interested in ways tables of results can be done in the easiest possible way, so that I (and maybe others?) will routinely use knitr/RMarkdown to produce quick reports of descriptive statistics and analyses to share with collaborators.

Outlined in the table below is a list of all the tables I've tested with links to the relevant code. I've grouped the tables according to whether they are able to output results in html, docx or pdf.

But hang on, I can make any table I want and output it to pdf(LaTex) - true, but the tables detailed here are not intended to be beautiful, publication ready tables. They are simple ways to make tables quickly.
